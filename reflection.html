1. Explain the concept of Responsive Design. What does the term mean? What did you discover in your research? Responsive Design is when you use HTML and CSS to make a website nicely fit into any browser or device regardless of size. A website can be resized, shrunk, hidden, enlarged to be more mobile friendly and compatible to devices such as phones, tablets and laptops. During this assignment and during my research I learned that you can use media queries to define diffrent styles and settings for diffrent browser sizes. For example a media querie would look like this: @media screen and (min-width:320px).With these CSS declarations, it is only for devices with those height and width properties. I also learned that media queries are not the best option to us as it can be difficult to work with them as you are defining specific sizes, but for this assignment, I found that it worked well.

2. How does responsive design relate to web technologies? How can web pages take advantage of it?In order for web technologies to work successfully in communication and ease of efficiency, it should be responsive to avoid collisions within the web. Web pages can take advantage of this by implementing a responsive design.
